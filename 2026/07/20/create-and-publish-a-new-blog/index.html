create and publish a new blog

Created2026-07-20|Updated2026-07-20
|Post Views:

1、 进入hexo根目录
cd D:\博客\test>

2、初始化 个人博客
hexo init

3、创建新博客
hexo new ‘new-blog’

4、生成静态网页
hexo g

5、预览
hexo s

6、发布到github
hexo d
